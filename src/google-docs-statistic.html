Google Docs Statistic

All statistics inserted via tidystats in Google Docs are tagged with a hyperlink. This hyperlink contains the identifier of the given statistic. It is thanks to this identifier that a Google Docs statistic remains updatable: when you upload a new file with changed statistical values for the same test, all tagged values will be updated with the corresponding new values (if any).

If you want the Google Docs statistics to remain updatable, make sure to keep the links in tact.

If you no longer wish for the Google Docs statistics to be updatable, you can freely remove the links.
