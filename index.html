tidystats

Reproducibly report statistics

What is tidystats?

tidystats is a software solution designed to help researchers report the output of statistical analyses. It consists of an R package whose main function is to gather the statistics from statistical analyses and a Microsoft Word add-in to reproducibly report statistics.

R package

The tidystats R package is available on CRAN. Install the package using R by running install.packages("tidystats"). For more information about the package, including documentation for the functions and tutorials on how to use tidystats, click the button below.

Microsoft Word add-in

The tidystats Word-addin is currently in development. It will soon be available in the Office Add-in Store. Once released, you will be able to able add statistics to your document using the file created with the tidystats R package.
